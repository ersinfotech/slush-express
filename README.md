slush-express
=============

ERS express generator used gulp

Usage
-----

```shell
$ npm i -g slush
$ npm i -g @ersinfotech/slush-express
$ mkdir ~/awesome-app
$ cd ~/awesome-app
$ slush express
$ npm i
$ npm start
```

Enjoy it

Upgrade
-------

```shell
$ npm i -g @ersinfotech/slush-express@latest
```
